# cashback-tracker-webapp
Track and manage your cashback offers in one place. An intuitive web app to replace that bulky [cashback tracking spreadsheet](https://medium.com/google-sheets-geeks/cashback-tracker-free-google-sheets-template-5fb845bb11ed?sk=42becf9288b466ea172ce5543565a4b0), built with Pug, Express, Bootstrap and MariaDB.
